# mindshare

`mindshare` is a Spring Boot 3 / Java 21 backend replica focused on the phase-one
scope of `auth`, `profile`, `storage`, `knowpost`, `feed`, and `search`.

## Current Status

Task 0 bootstrap is in place:

- Spring Boot 3 project skeleton
- Maven dependency management for phase-one modules
- Base application configuration
- Placeholder schema file
- Bootstrap context test

## Local Test

```powershell
mvn -q -Dtest=MindShareApplicationTest test
```
